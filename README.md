# MVPygame: A Simple MVP Framework for Pygame

## Demo

Setup the environment:

```sh
git clone https://github.com/Microwave-WYB/mvpygame.git
python -m venv .venv
source .venv/bin/activate
pip install .
```

### Dino Game

![Dino Game](/assets/dino.png)

```sh
python dino.py
```

### Flappy Bird

![Flappy Bird](/assets/flappy.png)

```sh
python flappy.py
```
